Tailwind CSS Box Model Example

Box Model

This is demostrates the properties of the box model

Padding

This box has p-4, which means padding of 1 rem(16px)

Border

This box has border-4, a dashed border with border-blue-300.

Margin

This box is separated from others using mb-4(margin-bottom)

Width & Height

This box has w-64(16rem) and h-32(8rem)
